# Excel Sales Analysis Project (update)


## Overview
This project analyzes a large sales dataset (500+ rows) using Microsoft Excel.

## Sheets
- Sales_Data: raw sales data
- Summary: core Excel formulas (SUM, AVERAGE, MAX, MIN, COUNTIF, SUMIF)
- Pivot_Analysis: aggregated sales analysis by region and category

## Excel Skills Used
- Excel Tables
- Calculated columns
- SUM, AVERAGE, MAX, MIN
- COUNTIF, SUMIF
- Pivot-style analysis
